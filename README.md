# Inbox Triage

A machine learning project for automatically classifying and prioritizing emails or text messages.

## Overview

Inbox Triage uses a trained ML model to categorize incoming messages, helping users manage large volumes of text by identifying importance or intent.

## Features

- Text classification using machine learning
- Training and evaluation datasets included
- Model persistence via JSON
- Command-line evaluation tools

## Project Structure

- `Models/` — Trained model files
- `data/` — Training and testing datasets
- `nb_model.py` — Model training logic
- `evaluate.py` — Model evaluation
- `IO_Utils.py` — Input/output utilities

## Dataset

- `train.tsv` — Training samples
- `test.tsv` — Evaluation samples

## Use Cases

- Email triage automation
- Message prioritization
- NLP classification learning projects

## Requirements

- Python 3.x
- Machine learning libraries (e.g. scikit-learn)

## Educational Focus

This project is well-suited for learning:
- Text preprocessing
- Supervised classification
- Model evaluation workflows
